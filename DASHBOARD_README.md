# DMT vs TFK Thickness Analysis Dashboard

## Overview
Interactive Dash web application for analyzing thickness measurements between DMT and TFK tools with comprehensive visualizations and statistical analysis.

## Features
1. **Site-by-Site Analysis**: Raw thickness trends at each measurement site with delta distribution
2. **Wafer Average Trends**: Average thickness per wafer for both tools with error bars and distributions
3. **Standard Deviation Analysis**: Variability analysis within each wafer
4. **Summary Statistics**: Comprehensive statistical table with all key metrics

## Quick Start

### 1. Generate Data (if not already done)
```bash
python thickness_comparison_app.py
```

### 2. Launch Dashboard
```bash 
python launch_dashboard.py
```

### 3. View Dashboard
Open your web browser to: **http://127.0.0.1:8050/**

## Dashboard Components

### Summary Cards
- Total matched measurement pairs
- Number of unique wafers  
- Mean thickness delta (DMT - TFK)
- Delta standard deviation

### Chart 1: Site-by-Site Thickness Analysis
- **Top plot**: Raw thickness measurements for each site
  - Lines show DMT (solid) and TFK (dashed) for each wafer
  - X-axis: Measurement sites (ordered by coordinates)
  - Y-axis: Thickness in Angstroms
- **Bottom plot**: Box plot of thickness deltas across all measurements

### Chart 2: Wafer Average Thickness Trends  
- **Top plot**: Average thickness per wafer
  - DMT averages (blue line with error bars)
  - TFK averages (red dashed line with error bars)
  - Error bars show standard deviation within each wafer
- **Bottom plot**: Distribution of wafer averages

### Chart 3: Wafer Standard Deviation Analysis
- **Top plot**: Standard deviation within each wafer
  - Shows measurement consistency for each tool
  - Lower std dev = more consistent measurements
- **Bottom plot**: Distribution of standard deviations

### Summary Table
Detailed statistics for each wafer including:
- DMT/TFK mean and standard deviation 
- Delta statistics (mean, std, min, max)
- Number of matched points
- Maximum matching distance

## Data Requirements
- **Input**: `matched_thickness_data.csv` (generated by `thickness_comparison_app.py`)
- **Columns**: WaferID, DMT_X_mm, DMT_Y_mm, TFK_X_mm, TFK_Y_mm, DMT_Thickness, TFK_Thickness, Distance_mm, Thickness_Delta, DMT_File, TFK_File

## Technical Details
- Built with Dash and Plotly
- Responsive design with interactive plots
- Real-time data filtering and sorting in summary table  
- All statistics calculated dynamically from source data

## Troubleshooting

### Dashboard won't start
- Ensure `matched_thickness_data.csv` exists (run the main analysis first)
- Check that all required packages are installed: `pip install -r requirements.txt`

### No data in charts
- Verify the CSV file contains data with the expected column names
- Check for any error messages in the terminal

### Browser won't open dashboard
- Manually navigate to: http://127.0.0.1:8050/
- Try a different port if 8050 is in use: modify `port=8050` in `thickness_dashboard.py`
